# Signal Bot

Telegram-бот для AI-дайджеста новостей. Работает через Tavily Search API.

## Переменные окружения (Railway)

| Переменная | Описание |
|---|---|
| `TELEGRAM_TOKEN` | Токен бота от @BotFather |
| `TAVILY_API_KEY` | API ключ с https://app.tavily.com |

## Деплой

GitHub → Railway → автодеплой при пуше в main.

## Как работает

1. Пользователь добавляет темы (до 10 штук)
2. Выбирает тему → бот делает запрос к Tavily Search
3. Tavily возвращает 5 свежих статей за последние 7 дней
4. Бот отправляет дайджест с заголовками, описаниями и ссылками
